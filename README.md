# DashboardEscunasPirata
Dashboard de visualização ao vivo dos bilhetes vendidos no dia

# Requisitos

- Python 3.11.16

- Beautiful Soup Elixir and Tonic 

- Requests HTTP Library

```
pip install bs4
pip install requests
```
